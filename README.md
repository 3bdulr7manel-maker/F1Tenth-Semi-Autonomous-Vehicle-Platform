# F1Tenth-Semi-Autonomous-Vehicle-Platform
Designed and developed a 1/10-scale semi-autonomous vehicle inspired by the F1Tenth platform. The project integrates a custom six-bar Ackermann steering system, custom PCB, and FreeRTOS-based dual-core control architecture, achieving precise motion control, obstacle avoidance, and reliable real-time performance.
